# AuraSmartDeskLamp

This is a landing page I created, it showcases a fictional smart desk lamp designed to automatically adjust it's brightness and warmth throughout the day to help reduce eye strain and keep you focused.

## Customization

To rebrand this template quickly, edit the centralized tokens and configuration:

- Visual tokens: open `style.css` and update the `:root` block at the top. Key tokens:
	- `--primary` / `--primary-rgb` — brand color
	- `--text` / `--text-rgb` — primary text color (also used for dark backgrounds)
	- `--surface` — surface/background color
	- `--muted` — muted text
	- `--border`, `--shadow-sm`, `--radius-md` — layout helpers

- Runtime config: open `main.js` and update the `CONFIG` object. Useful keys:
	- `CONFIG.currency` / `CONFIG.locale` — currency formatting
	- `CONFIG.prices` — product prices
	- `CONFIG.images` — product image mapping

Edit these two places and the site will follow the new brand quickly.
